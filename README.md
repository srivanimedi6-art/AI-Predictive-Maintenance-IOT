# 🔧 AI-Powered Predictive Maintenance for IoT Devices

## 📌 Overview
This project is an AI-based system that predicts machine failures using IoT sensor data such as temperature, vibration, and pressure.

## 🚨 Problem Statement
In industries, unexpected machine failures lead to:
- Downtime
- Financial losses
- Reduced efficiency

## 💡 Solution
This project uses Machine Learning to predict failures before they happen, enabling proactive maintenance.

## 🧠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## ⚙️ Features
- Sensor data simulation
- Failure prediction (0 = Safe, 1 = Failure)
- Accuracy & classification report
- Data visualization graph
- Model saving using joblib

## 📊 Workflow
Sensor Data → Preprocessing → Model Training → Prediction → Alert
